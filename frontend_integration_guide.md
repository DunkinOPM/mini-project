# Frontend Integration Guide for InsightEd API

This guide provides instructions on how to integrate your React/Next.js frontend with the newly upgraded InsightEd backend, which now utilizes the Google Gemini API for robust AI analysis.

## Overview

The new backend exposes a unified endpoint `POST /analyze` that handles the entire pipeline:
1. Video transcription
2. Slide/PDF text extraction
3. Hybrid RAG-based alignment mapping
4. Rich concept annotation generation

## 1. Connecting to the API

You can call the `/analyze` endpoint using standard `fetch` or `axios`. Since it expects files, you must use `FormData`.

```javascript
// Example using fetch in a React component
async function analyzeLecture(videoFile, documentFile) {
  const formData = new FormData();
  formData.append('video', videoFile);
  formData.append('document', documentFile);

  try {
    const response = await fetch('http://localhost:8000/analyze', {
      method: 'POST',
      body: formData,
      // Do NOT set Content-Type header manually when using FormData
    });

    if (!response.ok) {
      throw new Error(`API error: ${response.statusText}`);
    }

    const data = await response.json();
    console.log("Analysis complete:", data);
    return data;
  } catch (error) {
    console.error("Failed to analyze lecture:", error);
    throw error;
  }
}
```

## 2. Handling the Response Data

The API returns a JSON object with two key arrays: `mappings` and `annotations`.

### `mappings` (Timeline Markers)
Each mapping links a video timestamp to a specific slide/page.

**Example Item:**
```json
{
  "segment_id": 0,
  "timestamp": "02:30",
  "start": 150.0,
  "end": 165.0,
  "text": "Dynamic programming avoids recomputation...",
  "slide_id": 5,
  "concept": "slide",
  "confidence": 0.95
}
```

**UI Integration:**
- Render a video player component (e.g., `<video>` or `react-player`).
- Create clickable timeline markers below the video. When a marker is clicked, update the video player's `currentTime` to `mapping.start`.
- Automatically highlight the corresponding slide (e.g., `slide_id: 5`) in your document viewer when the video reaches `mapping.start`.

### `annotations` (Rich Explanations)
Each annotation contains a list of rich concepts generated by Gemini.

**Example Item:**
```json
{
  "timestamp": "02:30",
  "start": 150.0,
  "end": 165.0,
  "concepts": [
    {
      "concept": "Dynamic Programming",
      "explanation": "An algorithmic technique for solving an optimization problem by breaking it down into simpler subproblems.",
      "importance": "high"
    }
  ]
}
```

**UI Integration:**
- Create an "Insights" sidebar next to the video.
- As the video plays, filter the `annotations` array to find the annotation where `currentTime >= annotation.start && currentTime <= annotation.end`.
- Display the concepts and their explanations dynamically as they are being discussed. You can use color-coding based on the `importance` level (e.g., red for high, yellow for medium).

## 3. State Management Recommendations

If you are using React, we recommend using a global state manager (like Context API, Redux, or Zustand) to sync the video player, document viewer, and annotations.

```javascript
// Example Zustand Store
import { create } from 'zustand';

const useLectureStore = create((set) => ({
  currentTime: 0,
  mappings: [],
  annotations: [],
  currentSlideId: null,
  activeConcepts: [],
  
  setCurrentTime: (time) => set((state) => {
    // Find active slide
    const activeMapping = state.mappings.find(m => time >= m.start && time <= m.end);
    // Find active concepts
    const activeAnnotation = state.annotations.find(a => time >= a.start && time <= a.end);
    
    return {
      currentTime: time,
      currentSlideId: activeMapping ? activeMapping.slide_id : state.currentSlideId,
      activeConcepts: activeAnnotation ? activeAnnotation.concepts : []
    };
  }),
  
  setAnalysisData: (data) => set({
    mappings: data.mappings,
    annotations: data.annotations
  })
}));
```

By connecting your `<video onTimeUpdate={(e) => setCurrentTime(e.target.currentTime)}>` to this store, your UI will perfectly sync the slides and AI explanations with the video playback.
